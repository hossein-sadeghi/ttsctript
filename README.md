# ttsctript


### Install :
```bash
git clone https://github.com/Dr-Cyb3r/ttsctript.git
cd ttscript ; ./Installer.sh
```

### Usage : 
```bash
python3 ttscript.py
```

### Menu :
- password maker
- whois
- port scanner
- ip scanner
- proxy finder
- ip info
